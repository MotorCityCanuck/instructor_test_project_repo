# Medallion Design

**Purpose:** This document records how the team plans to use Raw, Bronze, Silver, and Gold layers in the NAPA Olympic Analytics Platform. The template provides structure only; teams must define the content, quality expectations, and evidence associated with each layer.

## Layer Design Template

| Layer | Purpose | Expected content | Quality expectations | Milestone alignment | Team notes |
|---|---|---|---|---|---|
| Raw | Source file landing or reference location. |  |  | Milestone 1 begins here. |  |
| Bronze | Source-aligned managed tables. |  |  | Milestone 1 evidence. |  |
| Silver | Cleaned and standardized business entities. |  |  | Milestone 2 evidence. |  |
| Gold | Decision-ready analytical products. |  |  | Milestone 3 evidence. |  |

## Notes for Teams

- Define each layer in business as well as technical terms.
- Keep quality expectations explicit.
- Align milestone evidence to layer maturity.
