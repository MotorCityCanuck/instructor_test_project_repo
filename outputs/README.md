# Outputs Guide

**Purpose:** This document explains how the outputs folder should be used to organize generated evidence and analytical artifacts for the NAPA Olympic Analytics Platform. The template provides folders only; teams must decide which outputs to generate, retain, and summarize for review.

## Subfolder Purposes

- `validation/`: quality-check summaries, reconciliation evidence, and other validation artifacts.
- `scorecards/`: small review-ready exports or summaries related to team-designed scorecards.
- `rankings/`: ranking outputs or excerpts suitable for review.
- `tournament_candidates/`: candidate-selection outputs or summaries generated by the team.
- `figures/`: screenshots, charts, or lightweight presentation visuals.

## Commit Guidance

Large generated outputs should not be committed unless specifically required and small enough for review. Prefer small summary exports, screenshots, or clearly labeled evidence artifacts when appropriate.
