# Notebook Guide

**Purpose:** This document explains the intended notebook sequence for the NAPA Olympic Analytics Platform template. The notebook files in this repository are outlines only; student teams must design, implement, validate, and document their own notebook logic and supporting evidence.

## Notebook Sequence

1. `01_ingest_raw_to_bronze.py`: plan and document source ingestion readiness.
2. `02_build_silver_entities.py`: design cleaned and standardized business entities.
3. `03_apply_data_quality_rules.py`: define and evaluate quality rules.
4. `04_build_gold_products.py`: design decision-ready analytical products.
5. `05_executive_analytics.py`: assemble executive-facing evidence and support tables.

## Working Expectation

These notebook files are Databricks-style outlines with headings, prompts, and TODO sections only. Teams should replace the placeholder structure with their own implementation and preserve evidence from each milestone run.
